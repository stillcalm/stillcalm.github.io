新建vue项目

Posted on 2021-10-01 Edited on 2021-10-21

首先安装Vue-Cil

  1. npm install vue-cli -g //全局安装 vue-cli

通过CIL构建应用

  1. 使用以下命令进行项目的建立，通过CIL工具初始化一个以Webpack为模版、项目名称为test的项目
  2. 此时项目是一个未经npm安装的项目，所以先通过cd命令进入该项目的根目录，然后使用npm install命令安装项目需要的插件
  3. 使用npm run dev命令运行项目，并在浏览器中打开loaclhost:8080查看效果

通过Vue UI构建应用

  1. 用vue ui创建项目时，应在cd node.js所在的系统盘。再输入vue ui，打开可视化界面，否则在创建项目时会产生错误

  2. 在选择合适的项目地址，调用适配的插件后，就可以点击新建项目了。在等待一小会后，项目就创建好了
